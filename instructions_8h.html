◆ construirInstrucoes()

Instrucao * construirInstrucoes ( void  )

Funcao que cria um vetor de instruções e as configura.

  • Após criar um vetor de instruções, a função define cada instrução com os campos corretos (opcode, numarg e numop), retornando o vetor no final.
  • OBS.:
    • Instruções com 'numarg' definido como -1 são instruções com numero de argumentos variáveis, e serão tratadas posteriormente.
    • Instruções com 'numop' definido como -1 são instruções que precisam verificar o descritor para definir seu número de operandos.

Retorna ponteiro para o vetor de instruções já configuradas
